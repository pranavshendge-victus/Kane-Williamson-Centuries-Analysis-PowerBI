# Kane Williamson International Centuries Analysis – Power BI Dashboard

This project analyzes the international cricket centuries scored by Kane Williamson using Power BI.

## Dashboard Features
- Total Runs KPI
- Total Centuries KPI
- Highest Score
- Average Score
- Centuries by Batting Position
- Runs by Ground
- Opponent Filter

## Tools Used
- Power BI
- Excel
- Data Cleaning
- DAX

## Dataset
International century records of Kane Williamson.
